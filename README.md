# Bingo Balls Caller

A mobile-first bingo ball caller with a large animated ball, called-number tracking, raffle-style sound effects, and browser speech announcement for each called letter and number.

## Features

- Two-button flow: `Ready to play` changes to `Next Call`, plus `Reset`
- Random 3-6 second flipping animation before each call
- Tracks all called B-I-N-G-O numbers
- Highlights the latest called number
- Uses built-in browser speech when available
- Single-file app, no build step needed

## Run Locally

Open `index.html` directly in a browser.

For phone testing from your computer, you can also publish it with GitHub Pages or send the `index.html` file to your phone and open it in Safari or Chrome.

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. Go to repository `Settings` > `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose the `main` branch and `/root`.
6. Open the Pages URL on your phone.
